# Task Manager Python

## Description
Command-line Task Manager to manage homework and study tasks.
This program allows students to add, view, edit, delete, mark tasks as done, and save/load tasks from a JSON file.

## Features
- Add new tasks
- View all tasks with status (Done / Not done)
- Edit existing tasks
- Delete tasks
- Mark tasks as done
- Save tasks to `tasks.json`
- Load tasks from `tasks.json`

## How to Run
1. Make sure Python 3 is installed.
2. Run the program:
   ```bash
   python tasks.py
